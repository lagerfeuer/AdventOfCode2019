# Advent Of Code 2019

## Dependencies

```bash
python3 -m venv venv/
source venv/bin/activate
pip install -r requirements.txt
```

## Running a solution

```bash
cd dayXX/
./main.py 1 # part 1
./main.py 2 # part 2
```

## Testing

```bash
pytest
```

or

```bash
cd dayXX/
pytest
```

### Issues

If `pytest` is not working, try `python -m pytest` or `deactivate && source venv/bin/activate` and run `pytest` again.
